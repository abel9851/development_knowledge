- 커맨드 라인을 rust로 작성한다.
- rust의 공식문서를 읽는것에 대해 회의감이 있었기 때문에 러스트로 커맨드라인을 작성해보는 것으로 돌파하려 한다. 
- rust는 누구나 안정적이고 효율적인 소프트웨어를 만들수 있도록 힘을 실어주는 언어다.
- rust는 오브젝트 지향언어는 아니지만 다양한 타입을 내포할수 있는 struct와 타입의 행동을 정의한 trait이 있다.
	- 상속은 할수 없지만 struct(구조체)는 다른 struct를 인자로 받아서 사용할 수 있다.
	- trait가 다른 trait를 사용하고자 한다면 "이 행동을 하려면 저 행동도 할 줄 알아야 한다"는 식으로 의존 관계(Bound)를 정의한다.
```rust

// 1. 걷는 행동 정의
trait Walk {
	fn walk(&self);
}

// 2. 뛰는 행동 정의 (뛰려면 걸을 줄도 알아야 함)
trait Run: Walk {
	fn run(%self);
}

// Run trait가 Walk를 상속받는 것처럼 보이지만, 데이터(프로퍼티)를 물려받는 게 아니라
// "내가 Run을 구현하고 싶다면, Walk 기능도 반드시 내 손으로 구현해 놔야 한다"는 
// 규칙만 강제하는 것이다.
```
