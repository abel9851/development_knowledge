## [ID: 202601242150] Two Pointers 패턴과 상태 공간 압축

**태그:** #algorithm #two-pointers #optimization #state-space #problem-solving

### 본문

투 포인터(Two Pointers)는 선형 데이터 구조에서 두 개의 독립적인 인덱스(상태)를 운영하여 탐색 효율을 극대화하는 알고리즘 패턴.

1. **상태 공간 압축 (State Space Reduction):**
    
    - 배열 내 두 요소의 모든 조합 $(i, j)$를 탐색할 경우 상태 공간은 $O(n^2)$임.
        
    - 데이터의 특성(대칭성, 정렬 순서 등)을 활용해 정답 가능성이 없는 조합을 즉시 제거함으로써 탐색 범위를 $O(n)$으로 압축함.
        
    - 예: Reverse String에서 $i$의 짝은 반드시 $n-1-i$임을 알기에 다른 조합을 확인하지 않음.
        
2. **독립적 상태 관리:**
    
    - 두 포인터는 각자의 진척도(Progress)를 담당함.
        
    - **수렴형 (Converging):** 양 끝에서 중앙으로 이동 (Reverse, Palindrome). 대칭 구조 활용.
        
    - **진격형 (Moving in same direction):** 서로 다른 속도로 전진 (Subsequence, Cycle Detection). 상대적 순서나 속도 차 활용.
        
3. **패턴의 특징:**
    
    - **결정론적 이동:** 매 단계에서 어떤 포인터를 움직일지 명확한 규칙이 존재함.
        
    - **단방향성:** 포인터가 뒤로 돌아가지 않음(Monotonicity). 이 특징이 중복 탐색을 방지하고 $O(n)$ 시간 복잡도를 보장함.
