`301 Moved Permanently` 리디렉션 상태 응답 코드는
요청한 리소스가 `Location` 헤더에 주어진 URL로 완전히 옮겨졌다는 것을 나타낸다. 브라우저는 이 페이지로 리다이렉트하고 검색 엔지는 해당 리소스로 연결되는 링크를 갱신한다.
